# 星火活动 App 上线说明

星火活动 App 是一个 vibe coding 做出来的活动报名和社群互动产品，准备明天公开上线。

## 当前功能

- 用户输入手机号后，调用 `/api/send-code` 获取短信验证码。
- 登录用户可以在活动详情页发表评论，评论会立即显示在公开页面。
- 主办方可以上传活动封面图，图片上传后会作为公开封面展示。
- 活动标题和地点输入后，可以调用 `/api/generate-description` 生成活动介绍文案。
- 管理员入口还在 demo 模式，前端 URL 加 `?admin=true` 后显示后台面板。
